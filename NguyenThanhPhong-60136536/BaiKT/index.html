What is Ajax?

As defiend in Wikipedia

AJAX is a group of interrelated web development techniques used to create interactive web applications or rich Interner applications. With Ajax, web applications can retrieve data from the sever asynchronosly in the background without interfering with the display and behavior of the existing page.
